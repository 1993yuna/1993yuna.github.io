Install Theme

To install the theme, please log in to your Shopify store and Click Online Store > Themes. From there, click on Upload Theme and upload the .zip file from the 'upload' folder included with this package

If your store is currently active and is using a different theme, you select button Publish theme what you have just upload

Shop Settings

Payments

1. From your shop admin, go to the Settings / Payments

2. Choose a payment gateway to accept payments for orders. Than checkout page is work

Checkout

1. From your shop admin, go to the Settings / Checkout

2. Select Accounts are optional

3. Click save

Add Page Wishlist

1. In a Online Store / Pages, find and click button "Add a page"

Product Desription

You have 3 short codes.
You could use Editor mode or Show HTML mode

Product description short code

Product description short code in Product description tab

1. [short_description]Here add your product short description[/short_description] - work on Product page below the product condition area , List product list view page, Search page and quick view

2. [product_description]Here add your product full description[/product_description] - work on Product page in product description tab
